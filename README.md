## File structure

adventofcode/year/day/part

```
adventofcode/
|-- year/
|   |-- day/
|       |-- input.txt
|       |-- README.md
|       |-- part/
|           |-- example.txt
|           |-- main.go
```

The input is inside the day directory since both parts use the same input. The examples are usually part specific so those are inside the part folder.
